# FPGA OpenCores I2C ADT7420 Temperature Monitor

Proyecto en Verilog para validar el funcionamiento del núcleo OpenCores I2C Master mediante comunicación con el sensor de temperatura ADT7420 integrado en la Nexys 4 DDR.

La temperatura se lee por I2C, se convierte a BCD y se muestra en el display de 7 segmentos de la FPGA.
